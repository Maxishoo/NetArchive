# Описание
***NetArchive*** - мобильное приложение для поддержания слабых связей. Многие специалисты теряют ценные связи из-за отсутствия удобного инструмента для регулярного поддержания контактов: они забывают важные детали общения, пропускают поводы для взаимодействия и упускают карьерные возможности. Приложение решает эту проблему с помощью метода «картотеки» — фиксации взаимодействий, системы проактивных напоминаний и интеграции с сервисами планирования, такими как Google Calendar.

Идея приложения заключается в том, чтобы превратить случайные знакомства в стратегические долгосрочные связи. В отличие от традиционных сервисов для хранения контактов, таких как Google Contacts или LinkedIn, которые выполняют в основном функцию пассивного хранения информации, приложение предлагает системный подход к управлению профессиональными отношениями.

Дополнительным преимуществом является встроенный ИИ-ассистент, который анализирует записи о контактах и предлагает подсказки для содержательного общения. Данный подход позволит повысить вовлечённость пользователей в системное развитие своей сети и улучшить качество профессионального нетворкинга.
